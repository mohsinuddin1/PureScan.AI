  <h3 className="text-xl font-bold text-slate-900 mt-8 mb-4">Hvorfor regnes retinoider, salisylsyre og kjemiske solfiltre som oksybenzon som usikre under graviditeten?</h3>
  <p>A-vitaminderivater som retinol, retinylpalmitat og tretinoin brukes mye i hudpleie mot aldring og akne. Orale retinoider er imidlertid kjent for å kunne gi fosterskader. Selv om absorpsjon gjennom huden er mindre, anbefaler hudleger og jordmødre å unngå alle retinoider under graviditet og amming for sikkerhets skyld. Salisylsyre under 2 % ved punktbehandling tåles vanligvis godt, men sterke kjemiske peelinger med BHA kan gi systemisk opptak. Kjemiske UV-filtre som oksybenzon penetrerer hudbarrieren, går over i morsmelk og virker hormonforstyrrende.</p>
  
  <h3 className="text-xl font-bold text-slate-900 mt-8 mb-4">Hvordan påvirker nitrater, syntetiske konserveringsmidler (BHA/BHT) og tungmetaller fosterets helse?</h3>
  <p>Kjøttpålegg og bearbeidet kjøtt inneholder ofte natriumnitritt og natriumnitrat for å beholde rosa farge og hemme bakterier. I kroppen kan nitrater omdannes til nitrosaminer, knyttet til økt oksidativt stress under svangerskapet. Syntetiske antioksidanter som BHA og BHT tilsettes for lengre holdbarhet, men kan passere morkaken. I tillegg kan tilsynelatende sunne produkter som proteinpulver eller viss sjømat skjule tungmetaller (bly, kvikksølv) som kan skade fosterets hjerneutvikling.</p>
  
  <h3 className="text-xl font-bold text-slate-900 mt-8 mb-4">Hvorfor er det usikkert å stole blindt på merking som 'Gravidsikker' eller '100 % Naturlig'?</h3>
  <p>Markedsføringsuttrykk som 'Gravidsikker' eller 'Naturlig' er ikke strengt regulert i lovverket. Produsenter kan trykke disse ordene på forsiden selv om produktet inneholder eteriske oljer som rosmarin eller salvie som kan stimulere livmorkontraksjoner eller øke blodtrykket. Tilsvarende inneholder 'naturlig' spekemat med selleripulver ofte mer nitrat enn tradisjonelt spekekjøtt. Man må alltid sjekke ingredienslisten nøye.</p>
